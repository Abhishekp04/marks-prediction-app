# 🎓 Student Marks Prediction Using Machine Learning

## 📌 Project Overview

This project predicts student marks based on academic and behavioral factors such as study hours, attendance, and sleep hours.

The project uses Machine Learning to estimate expected marks and provides an interactive Streamlit application for real-time predictions.

🌐 Live Demo: https://marks-prediction-app-1.streamlit.app/

---

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Joblib

---

## 📊 Features Used

### Input Features

- Study Hours
- Attendance
- Sleep Hours

### Target Variable

- Marks

---

## 🧹 Data Preprocessing

- Removed unnecessary columns
- Feature Selection
- Train-Test Split

---

## 🤖 Model Used

- Linear Regression

---

## 🎯 Prediction Goal

The model predicts student marks using:

- Study Habits
- Attendance Patterns
- Sleep Duration

---

## 🚀 Application Features

- Interactive Streamlit UI
- Real-time Marks Prediction
- Simple Student Performance Analysis
- User-Friendly Interface

---

## 💡 Key Insights

- Higher study hours generally improve marks.
- Better attendance positively influences performance.
- Sleep patterns can affect academic outcomes.
